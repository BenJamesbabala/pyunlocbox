=========
Tutorials
=========

The following are some tutorials which explain how to use the toolbox. They goes
in increasing degree of difficulty. If you have never used the toolbox before,
you are encouraged to follow them in order as they each build upon the other.

This module implements some demonstrations on how to use the toolbox to solve
real problems. The following demonstrations are included :

* :func:`compressed_sensing_1`: Compressed sensing problem solved by
  forward-backward proximal splitting algorithm.
